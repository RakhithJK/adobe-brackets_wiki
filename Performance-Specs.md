* Native OS keyboard settings: repeat rate and delay until repeat

* Size of viewport

    * Window/Editor size
* Line wrapping
* Syntax highlighting

    * Mixed content
* CodeMirror pollingInterval
* Number of inline widgets
* Number of keyboard shortcuts
* Electric chars
